# 🏅 Tokyo Olympics Data Engineering Pipeline

## 📌 Project Overview
Built an end-to-end data engineering pipeline using Azure Databricks and ADLS.

## 📂 Data Source
Data is stored in Azure Data Lake (ADLS Gen2).

Sample Olympic datasets used:
- Athletes
- Coaches
- Medals
- Teams
- EntriesGender
